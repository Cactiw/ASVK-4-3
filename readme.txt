 Дополнительно использовалась библиотека https://github.com/nlohmann/json для работы с json объектами в С++
 
 В Makefile присутствует цель test в случае, если необходимо автоматически применить тесты из файлов вида *D.json, где D - номер теста. В этом случае команда make test скомпилирует исполняемый файл, который может применить тесты к программе. В аргументах командной строки после запуска получившегося исполняемого файла следует указать путь к файлам с тестами, программа сама применит тесты, результат будет в tests_out.txt.
